# SmsClub API Client

Client for http://smsclub.mobi/ SMS gate. Allows you to send and retrieve status of sent SMS via XML API.

## Installation

Add this line to your application's Gemfile:

    gem 'sms-club'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sms-club
